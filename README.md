# Strompreis_Analyse
Analyse deutscher Großhandelsstrompreise (2021–2025) mit Python, Pandas und Matplotlib. Der Datensatz wurde bereinigt, aufbereitet und hinsichtlich Preisentwicklung, negativer Strompreise, Extrema, gleitender Durchschnitte und Tagesprofile untersucht. Ergebnisse wurden visuell dargestellt.
